Title                   : Vertex Relocation
Filename                : vx_reloc.wad
Release date            : 2025-12-17
Author                  : 4chan.org/vr/doom
Email Address           : [redacted]

Description             : 33 maps built around vanilla Doom 2 enemy placement.

                          Mappers were allowed to change anything that isn't
                          a monster, but were not allowed to move the enemies 
                          or modify their flags. Despite the name, there were
                          no restrictions on map geometry, so few (if any)
                          vertices are shared with the original maps.

                          The mapset is generally balanced for pistol starting.

                          vx_reloc_4x3.wad contains HUD graphics for engines
                          that don't support widescreen status bar (Odamex etc)

                          doom-sat.pdf contains details on implementation of a
                          boolean satisfiability puzzle using the Doom engine.
